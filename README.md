## Tests
**Test Everything ([PHPUnit](https://phpunit.de))**

`./vendor/bin/phpunit`

**Run Mutation Tests ([Infection](https://infection.github.io/))**

`./vendor/bin/infection`

**Run Static Analysis ([Psalm](https://psalm.dev/))**

`./vendor/bin/psalm`
